# 多行业数据分析与数据挖掘实战项目集

该项目集覆盖**电商、O2O服务、广告营销**等领域，基于Python数据科学栈（Pandas、Numpy、Matplotlib、Seaborn、Scikit-learn等），完成从**数据清洗、多维度分析到用户画像构建、聚类建模**的全流程实践，助力业务决策与运营优化。


## 核心项目模块

### 1. 电商用户行为与RFM画像分析
- 对电商交易数据进行**清洗治理**（重复值、空值、异常值处理），分析用户**重复购买行为、交易时间分布、区域退单率**等特征。
- 构建**RFM用户画像**，结合**优惠券评分模型**实现用户分层，同时设计“是否退单”“购买数量”等维度的评分体系，支撑精准营销决策。


### 2. O2O配送服务不满意率分析
- 运用**文本挖掘技术**（jieba分词、CountVectorizer）解析用户差评标签，定位“仪表不整”“送达超时”等核心不满因素。
- 从**配送时长、站点、骑手**维度开展多维度分析，通过柱状图、热力图等可视化手段，识别高差评站点、骑手及关键差评原因，为服务流程优化提供方向。


### 3. 广告效果多维度分析与用户聚类
- 整合广告曝光、广告特征、用户画像数据，从**产品维度**（adgroup_id、价格区间、品类）和**用户维度**（性别、年龄、购物层级）分析广告点击率（clk）分布规律。
- 引入**卡方检验**验证用户特征与点击率的关联性，运用**KMeans聚类算法**（结合轮廓系数选优）实现用户精准分群，为广告策略优化提供数据支撑。


## 技术栈
- 数据处理：Pandas、Numpy；
- 可视化：Matplotlib、Seaborn；
- 文本挖掘：jieba、Scikit-learn（CountVectorizer）；
- 建模分析：Scikit-learn（KMeans、StandardScaler）、统计检验（卡方检验）；
- 业务方法：RFM模型、用户分层评分等。


## 应用场景
- 电商领域：用户行为洞察、RFM分层运营、精准营销；
- O2O服务：差评归因分析、服务流程优化、骑手与站点管理；
- 广告营销：广告效果评估、用户精准分群、投放策略优化。


该项目集可作为数据科学初学者的实战参考，也可为各行业从业者提供数据分析方法论与场景化解决方案的借鉴。
# 金融信贷用户画像与风险分析项目

该项目基于信贷业务数据集，通过Python数据科学工具栈（Pandas、Numpy、Matplotlib、Seaborn等），从**用户画像、资金储备、逾期风险、利率策略**等维度开展深度分析，为信贷业务的客群运营、风险管控和产品设计提供数据支撑。


## 核心分析模块

### 1. 用户画像分析
- **性别维度**：男性在平台贷款占比达67.6%，但男女平均贷款金额差异仅约2%，说明性别对贷款金额需求影响较小。
- **新老客维度**：老客户占比73%，平台用户粘性较高；新客平均借款水平（约5686元）远高于老客，且新客中男性占比更高，但从新客内部看性别影响有限。
- **年龄维度**：借款用户以中青年为主（33岁以下占比超75%），15-30岁新客平均贷款水平高于老客，该年龄段新客潜力较大。


### 2. 资金储备分析
通过对借款成功日期和借款金额的时间序列分析，结合3σ原则测算出每日需准备的资金约为11814846元，为资金调度提供量化依据。


### 3. 逾期风险分析
- **评级/借款类型**：初始评级为e、f的用户逾期率呈断崖式上升（或因样本量少导致）；“app闪电”借款类型逾期率显著低于其他类型，建议加强与该类型合作。
- **性别/借款金额**：男性逾期率低于女性；借款金额2000元以下逾期率最低，2000-3000元区间逾期率最高，可考虑侧重小额贷款营销以降低风险。


### 4. 借款利率分析
女性对贷款利率的接受度低于男性，且女性逾期率高于男性，可针对女性定制差异化产品（如分期贷款、利率优惠等）。


## 技术栈
- 数据处理：Pandas（数据读取、清洗、分组聚合）、Numpy（数值计算）。
- 可视化：Matplotlib、Seaborn（饼图、柱状图、折线图等多维度可视化）。
- 分析方法：描述性统计、分组对比、时间序列分析、3σ原则等。


## 应用价值
- 助力信贷业务**客群精准运营**：明确高潜力客群（如15-30岁新客），针对性设计营销与产品策略。
- 强化**风险管控**：通过逾期率维度的多因素分析，优化借款类型选择、金额区间布局。
- 提升**资源配置效率**：量化资金储备需求，保障业务流畅运转。
